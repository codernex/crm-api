## Run Locally

Clone the project

```bash
  git clone https://github.com/Softronixs-System-Ltd/crm-backend.git
```

Go to the project directory

```bash
  cd crm-backend

```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
